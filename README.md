# Pixel & Pulse: CSS Art Collection

✨ **Interactive CSS Art Gallery by Agus Satria Adhitama** ✨

Welcome to **Pixel & Pulse**, a collection of **20+ unique and animated CSS artworks**, created purely with HTML and CSS. This gallery showcases fun, creativity, and interactive mini-animations - from Pikachu to Rockets, Music Notes, Flowers, and more ! 

---

## ⚡ Live Demo
https://agusadhitama.github.io/CSS-art-museum/

---

## 🎨 Features

- 20+ interactive **CSS art pieces**  
- Mini animations: bounce, spin, float, sway, heartbeat, twinkle, and more  
- Responsive **grid layout** for all screen sizes  
- Hover effects with shadow & transform for dynamic interaction  
- Fully **HTML + CSS**, no JavaScript required  
- Modern **dark theme** for immersive experience

## ⚡ Usage

1. **Clone the repository**  

```bash
git clone https://github.com/agusadhitama/pixel-pulse-css-art.git
```
2. **Open index.html in your favorite browser**
```bash
open index.html
```
3. **Enjoy the interactive CSS gallery!**
